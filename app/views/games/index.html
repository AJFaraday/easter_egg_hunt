Easter Egg Hunt API

This application will hide an Easter egg in a garden, then accept API calls to guess where the egg is. It'll tell you if your X or Y coordinate is higher or lower than the egg's position. It'll also tell you if either or both of your co-ordinates are correct.

  • The garden has a size of 100 by 100. (The coordinates are from 1 to 100)
  • You can guess as many times as you like, but try to get it in the smallest number of turns.
  • The returned data is in JSON format.

The API calls

New Game

Example call:

  easter_egg_hunt.andrewfaraday.com/new_game

This will set up a garden and hide an egg in it. It will hand you a game_id, which is required for playing the game.

Example return:

  {game_id: 'abcde'}

Guess 




Example call:



  easter_egg_hunt.andrewfaraday.com/guess/abcde/50/50





  This will take a guess at where the egg is. The app will tell you if the
  actual position of the egg is higher or lower than your guess.




Example return:



  {
    x: 'higher',
    y: 'lower'
  }



The results for each coordinate can be:



  
higher

  
You need to guess higher.


  
lower

  
You need to guess lower.


  
right

  
You've got the right answer!
